# 🎓 ADS Mackenzie - Práticas e Exercícios

Este repositório foi criado para documentar minha jornada acadêmica no curso de **Análise e Desenvolvimento de Sistemas** (ADS) pela **Universidade Presbiteriana Mackenzie**. Aqui, concentro exercícios, laboratórios e projetos desenvolvidos ao longo dos semestres.

## 📌 Sobre o Curso
O curso foca na prática de formar profissionais capazes de analisar, projetar, documentar, especificar, testar, implantar e manter sistemas computacionais de informação.

## 🗂️ Estrutura do Repositório

O conteúdo está organizado por disciplinas para facilitar a navegação:

* **1º Semestre (2025/2):**
    * `ALGORITMO E PROGRAMAÇÃO I`: Exercícios fundamentais utilizando Python.
    * `FUNDAMENTOS DA WEB`: Primeiros passos com HTML, CSS e JavaScript.
 

## 🛠️ Tecnologias e Ferramentas
* **Linguagens:** Python.
* **Ambiente:** VS Code.
* **Versionamento:** Git e GitHub.

## 🚀 Objetivo
Utilizo este espaço como um portfólio para demonstrar a minha evolução em matérias de raciocínio lógico e a aplicação prática dos conceitos teóricos aprendidos em aula.
